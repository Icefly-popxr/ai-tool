# 概念抽取工作流

> 配合AI使用的自动化流程

---

## 📋 完整流程

```
你写好一张知识卡片
    ↓
你发送：概念抽取 [卡片ID]
    ↓
AI分析卡片 → 提取候选概念
    ↓
AI展示给你确认 ← 你在这里
    ↓
你回复：确认写入 [概念A][概念B]
    ↓
AI自动创建文件到 06 - Wikis 概念库/concepts/
```

---

## 💬 你发给AI的提示词

### 触发词：概念抽取

```
概念抽取

以下是我的知识卡片内容：

[粘贴卡片内容]

请完成以下步骤：
1. 提取核心概念（每个概念给出：名称、一句话定义、对应卡片内容）
2. 标注每个概念是否值得收录到Wiki库（判断标准：经典学术、跨学科可迁移、有价值）
3. 展示给我确认

确认后，请直接帮我创建概念卡片文件到 06 - Wikis 概念库/concepts/ 目录。
```

---

## ⚙️ AI写入的文件格式

当你确认后，AI会自动在 `06 - Wikis 概念库/concepts/` 目录下创建如下文件：

```markdown
---
date: 2026-05-09
source: [[卡片名称]]
type: concept
---

# 概念：[概念名称]

## 定义
[一句话定义]

## 来源
- 来自：[[卡片名称]]
- 日期：2026-05-09

## 核心要点
1. [要点1]
2. [要点2]

## 相关概念
- [[概念A]] - [关系]
- [[概念B]] - [关系]

## 应用场景
- [场景1]
- [场景2]
```

## 📝 实操示例

### 你发送：
```
概念抽取

🏷️ 卡片名称：频繁洗头可能加剧头皮屑
💡 核心洞察：每天洗头破坏头皮屏障，反而加剧头皮屑
💚 行动指南：降低频率到2-3天一次，用温和去屑洗发水
🟢 绿灯场景：每天洗头但头皮屑不见好转
🔴 红灯场景：有皮肤疾病需就医
```

### AI返回：
```
提取到以下候选概念：

1. 头皮屏障
   - 定义：头皮表面的保护层，过度清洁会破坏
   - 值得收录：✅（皮肤科跨认知科学）

2. 过度清洁
   - 定义：清洁频率超过皮肤自我修复能力
   - 值得收录：✅（可迁移到多个领域）

3. 自我修复周期
   - 定义：身体组织从损伤中恢复所需时间
   - 值得收录：⚠️ 较通用，建议简化

请确认是否写入Wiki库？

可回复：确认写入 头皮屏障、过度清洁
可回复：只写 头皮屏障
可回复：都不写
```

### 你回复：
```
确认写入 头皮屏障、过度清洁
```

### AI自动创建：
创建 `06 - Wikis 概念库/concepts/头皮屏障.md` 和 `06 - Wikis 概念库/concepts/过度清洁.md`
