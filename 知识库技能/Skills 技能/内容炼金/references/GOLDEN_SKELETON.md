# 黄金骨架7步详解

本文档详细说明口播逐字稿的7步黄金骨架结构、时长分配和撰写技巧。

---

## 整体时间参考

| 总时长 | 开场 | PROBLEM | 故事 | 观点 | 对标 | 行动 | 结尾 | 总计 |
|--------|------|---------|------|------|------|------|------|------|
| **3min** | 15s | 30s | 60s | 20s | 30s | 20s | 10s | ~185s |
| **5min** | 20s | 45s | 90s | 30s | 45s | 20s | 15s | ~265s |
| **7min+** | 25s | 60s | 120s | 40s | 60s | 25s | 20s | ~350s |

## 第1步：开场 (15-20秒)

### 目标
一句话抓住注意力，制造悬念或认知冲突。

### 4种开场方式

#### 方式A：反常识开场
```
"你以为[X]，但其实[Y]"
例："你以为努力就能成功？但努力是最廉价的东西。"
```

#### 方式B：问题开场
```
"你有没有想过[X]？"
例："你有没有想过，为什么你总是遇到渣男？"
```

#### 方式C：数字开场
```
"[数字]的人都不知道[X]"
例："90%的人都在用错误的方式减肥。"
```

#### 方式D：身份开场
```
"如果你也是[X]，那么[Y]"
例："如果你也是一个内向的人，你会发现......"
```

### 开场禁忌
- ❌ 不要自我介绍（除非必要）
- ❌ 不要说"今天给大家分享"
- ❌ 不要用陈词滥调（"随着社会的发展"）
- ❌ 不要讲太长，15-20秒必须进入正题

### 开场示例

**❌ 不好：**
> "大家好，我是XX，今天想跟大家聊聊自律这个话题，因为我觉得自律对大家很重要..."

**✅ 好：**
> "你知道吗，自律可能是你人生中最大的一个谎言——你越追求自律，你就越失败。"

---

## 第2步：PROBLEM (30-45秒)

### 目标
描绘现状困境，让观众产生共鸣，说出"这就是我"。

### 3层结构

```
第一层：困境是什么（现象）
第二层：为什么会有这个问题（原因）
第三层：不解决会有什么代价（后果）
```

### 写作技巧

**困境描述要具体：**
```
❌ "很多人都有拖延症，很痛苦"
✅ "每次准备开始工作，脑子里就开始放电影——要不要喝杯水？要不要刷一下手机？等到真的开始做，已经过了半小时"
```

**原因分析要深刻：**
```
❌ "因为你不自律"
✅ "因为你把'开始'想得太大了，大脑本能抗拒"
```

**代价要有冲击力：**
```
❌ "会影响工作效率"
✅ "结果是：白天假装很忙，晚上愧疚失眠，第二天继续恶性循环"
```

### PROBLEM时间控制

| 时长 | PROBLEM建议 |
|------|----------|
| 3min | 30秒，聚焦一个问题 |
| 5min | 45秒，可以拆成2个小问题（铺垫递进） |
| 7min+ | 60秒，深挖问题根源 |

### 铺垫递进技巧（5min+）

当PROBLEM需要讲深时，用"两个问题递进"：

```
第一个问题：[表面问题]
停顿1秒
第二个问题：[更深层问题]
让听众体验"认知逐层打开"
```

**示例：**
> "你是不是经常计划完不成？...（停顿1秒）...但你有没有想过，真正的问题不是你执行力差，而是你的计划从一开始就注定失败？"

## 第3步：故事/案例 (60-90秒)

### 目标
用具体故事证明你的观点，让抽象道理变得可感知。

### 故事结构

```
情境：[发生了什么] → 30%
转折：[但是] → 20%
行动：[我做了] → 30%
结果：[收获了] → 20%
```

### 好故事的要素

1. **有细节**：时间、地点、人物、具体的数字
2. **有冲突**：不是一帆风顺
3. **有情感**：你能感受到当时的情绪
4. **有转折**：突然发现了一个道理

### 故事来源优先级

```
✅ 自己的真实经历（最有说服力）
✅ 朋友/熟人的真实经历（可匿名）
✅ 公开报道/数据支撑
❌ 编造的故事
❌ 名人八卦（没有数据支撑的）
```

### 故事时长控制

| 时长 | 故事建议 |
|------|----------|
| 3min | 1个故事，60秒，讲完整 |
| 5min | 1个核心故事+1个简短案例 |
| 7min+ | 2个故事，或1个深度故事 |

### 故事示例（精简版）

```
我以前也是个重度拖延症患者。

有一次，老板让我周一交报告。
我周五开始想，周六准备写，周日打开电脑...
然后我开始整理桌面，整理了2个小时。

最后周一早上6点爬起来，通宵赶完了报告。
提交的那一刻，我就知道——这辈子我不想再这样了。

从那天起，我开始研究怎么"不靠意志力"也能开始工作。
结果发现，答案简单到可笑。
```

## 第4步：观点萃取 (20-30秒)

### 目标
一句话说出核心洞见，让观众记住。

### 观点萃取公式

```
不是[X]，而是[Y]
真正重要的是[X]
答案很简单：[一句话]
```

### 好观点的标准

- ✅ 一句话说清楚
- ✅ 有认知落差
- ✅ 可以直接指导行动
- ✅ 朗朗上口

### 观点萃取示例

```
❌ "我们要学会控制时间"（太抽象）
✅ "不是时间不够，是你把优先级排错了"（有冲突）

❌ "要保持积极的心态"（太鸡汤）
✅ "积极不是假装开心，是找到值得开心的理由"（有洞见）

❌ "要不断学习提升自己"（太泛）
✅ "学习不是为了知道更多，是为了能做不同的事"（有深度）
```

## 第5步：对标迁移 (30-45秒)

### 目标
让观众知道"这个道理不只在这里有用"，扩大应用范围。

### 3种对标方式

#### 方式A：跨领域类比
```
就像[X领域]一样
[这个领域]也是一样的道理
所以[Y]
```

**示例：**
> "就像健身一样，你不需要每天都练到力竭，
> 重要的是持续。
> 学习也是如此——不需要每天学10个小时，
> 只需要每天比昨天多学10分钟。"

#### 方式B：人群扩展
```
不只是[X人群]
[Y人群]也一样
因为本质上都是[共同点]
```

**示例：**
> "不只是职场新人需要学会汇报，
> 就连工作10年的老员工也经常犯这个错——
> 因为本质不是经验问题，是沟通逻辑问题。"

#### 方式C：场景迁移
```
在[X场景]要用[X方法]
在[Y场景]其实也一样
核心是[共同逻辑]
```

**示例：**
> "跟老板汇报要用'结论先行'，
> 写简历也一样——
> 面试官没时间看你的流水账，
> 他们只关心你能解决什么问题。"

### 对标迁移禁忌

- ❌ 不要超过3个类比（会乱）
- ❌ 不要用观众不熟悉的领域
- ❌ 不要为对标而对标

---

## 第6步：落地行动 (15-20秒)

### 目标
让观众"现在就能做"，降低行动门槛。

### 行动指南公式

```
你今天可以做的最小行动是[X]
成功标志是[Y]
不要等到[借口]，现在就[行动]
```

### 好行动的标准

- ✅ 足够小（5分钟内能开始）
- ✅ 足够具体（有明确动作）
- ✅ 有成功标志（知道做没做对）
- ✅ 有时间节点（什么时候开始）

### 行动示例

```
❌ "要每天坚持学习"（太模糊）
✅ "今天睡前花5分钟，列出你明天最重要的3件事——不需要多，就3件，写下来就够了"

❌ "要改变拖延习惯"（太复杂）
✅ "下次想拖延的时候，先把手机放到另一个房间——就这一步，不做别的"
```

---

## 第7步：结尾 (10-15秒)

### 目标
收尾有力，回应开场，形成闭环。

### 3种结尾方式

#### 方式A：回应开场
```
回到开头的问题——
现在你知道[X]
下次遇到[Y]的时候
记得[核心建议]
```

#### 方式B：反问收尾
```
所以，你现在愿意[X]吗？
还是你想像以前一样[Y]？
选择权在你手上。
```

#### 方式C：金句收尾
```
最后送大家一句话：[你的核心金句]
谢谢大家。
```

### 结尾禁忌

- ❌ 不要说"希望今天的分享对大家有帮助"（太套路）
- ❌ 不要突然换话题
- ❌ 不要拖太长

## 完整示例（3分钟版本）

```
【开场 15秒】
你知道吗，拖延症最大的谎言就是"等我有状态了再做"——
但真相是，你永远不会有那个状态。

【PROBLEM 30秒】
每次准备开始工作，脑子里就开始放电影：
要不要先收拾一下桌子？
要不要泡杯咖啡？
等到真的开始，已经过了半小时。
结果白天假装很忙，晚上愧疚失眠，第二天继续恶性循环。
问题不是你不自律，是你把"开始"想得太大了。

【故事 60秒】
我以前也是这样。
有一次周一要交报告，我周日早上10点打开电脑，
然后整理桌面用了2小时...
最后周一早上6点爬起来通宵。
提交的那一刻我就知道——我不想再这样了。

【观点 20秒】
所以，解决拖延不是靠意志力，
是靠把"开始"变得足够小。

【对标 30秒】
就像健身一样，你不需要每天都练到力竭，
重要的是动起来。
学习、工作都是一样的——
不需要准备完美，只需要开始5分钟。

【行动 15秒】
今天你可以做的：
打开手机闹钟，定一个5分钟后的提醒，
到时候不管在干嘛，立刻开始做那件你一直拖着的事。

【结尾 10秒】
所以，状态不是等来的，是做出来的。
今天的你，准备好开始了吗？
```

## 常见问题

### Q1: 如果没有故事怎么办？
**A**: 用数据/案例替代，或者用"朋友的故事"（可匿名）。核心是有具体细节，不一定要亲身经历。

### Q2: 对标迁移总是很牵强？
**A**: 说明这个对标不适合。换一个更贴切的，或者直接去掉对标环节，聚焦核心观点。

### Q3: 行动建议总是不够小？
**A**: 记住标准：5分钟内能开始。不断问"还能再简化吗？"，直到足够小。

### Q4: 开头总是讲不清楚？
**A**: 先写完整版，再回头删减前30%的内容。开场的每个字都要有存在理由。
