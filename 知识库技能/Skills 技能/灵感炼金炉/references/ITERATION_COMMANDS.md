# 迭代指令详解

本文档详细说明灵感炼金炉的三种迭代指令：「更深」「换角度」「更毒舌」。

---

## 「更深」

### 功能
在当前洞见上再挖一层，追问背后还有什么。

### 使用场景
- 当前洞见已经很好，但感觉还可以继续挖
- 用户想要更彻底的思考
- 第一层洞察是表象，想看本质

### 使用方法
用户输入：`「更深」` 或 `「再深一点」` 或 `「继续挖」`

### 内部流程
```
当前洞察 → 继续追问"为什么" → 再问一层"为什么这个是对的" → 找更根本的原因
```

### 示例

**原始洞见**：
> 🔥 破局点：不是读得不够，是读得太多，消化太少

**「更深」后的新洞察**：
> 🔥 破局点：你不是在读书，你是在用"读书"逃避"行动"
> 
> 展开：
> - 第一层：读书多但不行动，是因为行动会暴露问题
> - 第二层：读书是安全的——读再多也不会失败，但实践会
> - 第三层：你需要一个小到不可能失败的行动，而不是再读一本书

### 判断标准
「更深」成功的标志：
- ✅ 新洞察比原来的更根本
- ✅ 能解释原来的洞察为什么对
- ✅ 给出了更高维度的行动建议

---

## 「换角度」

### 功能
换一把刃重新切割同一主题，探索不同方向。

### 使用场景
- 当前洞见的方向不是用户想要的
- 用户想要更多元的视角
- 第一个洞见已经很好，但想看其他可能性

### 使用方法
用户输入：`「换角度」` 或 `「换一个」` 或 `「换个刀」`

**可选指定**：
- `「换角度，换悖论刃」` - 指定用某把刀
- `「换角度，不要类比刃」` - 排除某把刀

### 内部流程
```
当前使用的刀 → 切换到另一把刀 → 重新切割同一主题 → 新洞见
```

### 示例

**原始洞见**（溯源刃）：
> 🔥 破局点：你不自律，是因为不知道自己要什么

**「换角度」后**（用类比刃）：
> 🔥 破局点：自律像减肥——靠饿的人最终都会暴食，靠做好吃健康餐的人才能持续
> 
> 展开：
> - 第一层：靠意志力的自律就像靠饿减肥，早晚会崩溃
> - 第二层：真正能持续的，是把"痛苦的事"变成"喜欢的事"的系统
> - 第三层：找到一个让你不需要坚持也能做的微调

### 七把刀可以换出的洞见方向

| 刀 | 洞见方向 |
|---|---------|
| 🔴 悖论刃 | 找矛盾、破共识 |
| 🔴 杠杆刃 | 找关键、找支点 |
| 🟡 溯源刃 | 找根因、剥本质 |
| 🟢 反转刃 | 翻转前提、开新视角 |
| 🔵 类比刃 | 跨领域借结构 |
| 🟣 笨拙刃 | 剥包装、说真话 |
| ⚪ 尺度刃 | 大小尺度看变化 |

### 判断标准
「换角度」成功的标志：
- ✅ 提供了完全不同的视角
- ✅ 仍然锋利、有价值
- ✅ 不是浅层次的"换个说法"

---

## 「更毒舌」

### 功能
提升犀利度，加入讽刺和反讽，让洞见更尖锐。

### 使用场景
- 用户觉得当前洞见太温和
- 想要更有冲击力的表达
- 需要写成自媒体标题、口播脚本

### 使用方法
用户输入：`「更毒舌」` 或 `「更犀利」` 或 `「狠一点」`

### 内部流程
```
当前表达 → 找更刺激的说法 → 加入反讽/讽刺 → 保持核心洞察不变
```

### 注意事项
- ⚠️ 毒舌不是骂人，是更尖锐
- ⚠️ 不能改变核心洞察，只能改变表达方式
- ⚠️ 要保持善意，不是为了刻薄而刻薄

### 示例

**原始洞见**（温和版）：
> 🔥 破局点：追求自律本身就是最大的不自律

**「更毒舌」后**：
> 🔥 破局点：那些天天喊自律的人，其实是在大声宣布"我控制不住自己"
> 
> 金句（更毒舌版）：
> - 原版："自律是成功的第一步"
> - 毒舌版："自律是loser的遮羞布"
> 
> - 原版："坚持就是胜利"
> - 毒舌版："坚持是因为你没找到不坚持也能做的理由"

### 毒舌技巧库

#### 1. 把褒义词贬义化
```
努力 → 瞎忙
坚持 → 死撑
自律 → 自虐
上进 → 焦虑
```

#### 2. 撕掉高大上的包装
```
成功 → 运气好
选择大于努力 → 运气好的人这么说
做自己 → 自私的借口
```

#### 3. 说出潜规则
```
他很努力 → 他不敢停下来
她很优秀 → 她很焦虑
我很独立 → 我不敢依赖人
```

#### 4. 反讽结构
```
不是说"读书没用"，而是说"读再多书也不耽误你过不好这一生"
不是说"努力没用"，而是说"努力是最廉价的借口"
```

### 判断标准
「更毒舌」成功的标志：
- ✅ 保持核心洞察不变
- ✅ 表达更尖锐、更有冲击力
- ✅ 用户看完有"被戳中"的感觉
- ✅ 不变成纯粹的情绪发泄

---

## 指令组合使用

### 推荐组合

**深度挖掘**
```
「更深」 → 「更深」 → 「更毒舌」
```
先挖深，再让表达更尖锐

**多元探索**
```
「换角度」 → 「更深」 → 「更毒舌」
```
先换个角度，再挖深，最后犀利表达

**快速调整**
```
「更毒舌」 → 「换角度」
```
如果毒舌版不够好，换个角度重新来

### 使用频率建议
- 「更深」：当你想把一个洞见吃透时
- 「换角度」：当你想要更多可能性时
- 「更毒舌」：当你要产出内容时

---

## 迭代次数限制

### 建议次数
| 指令 | 建议次数 | 原因 |
|------|---------|------|
| 「更深」 | 2-3次 | 挖太深会脱离实际 |
| 「换角度」 | 2-3次 | 太多会迷失方向 |
| 「更毒舌」 | 1-2次 | 表达调优不需要太多轮 |

### 提示用户
如果用户迭代太多次，可以提示：
```
⚠️ 提示：已经迭代[X]次了，建议选择一个版本继续深挖。
或者换一个主题？
```

---

## 特殊指令

### 「回到最初」
用户输入后，回到最开始输出的版本

### 「就这个了」
用户确认满意，停止迭代

### 「我要整理一下」
用户暂停，回去消化，下次继续

### 「还有没有其他角度」
要求再输出一个不同方向的洞见（不需要迭代，直接输出新选择）

---

## 迭代后输出格式

### 完整迭代输出
```
## [迭代版本 X.X]

🔥 **破局点**
→ [当前版本]

🗡️ **展开**
→ 第一层：[...]
→ 第二层：[...]
→ 第三层：[...]

💬 **金句**
→ [一句话]

⚠️ **边界**
→ [失效场景]

---
📝 [迭代历史简述]
```
