# A股投研资料路由与分析

当用户分享资料（链接或文本）需要分析时，自动路由到合适的处理流程。

## 路由规则

### 1. 如果是链接（URL）→ 先调用 agent-reach 提取内容

- 提取内容中包含A股上市公司 → 按7列标准表格分析输出
- 提取内容中无上市公司 → 归纳总结，生成.md文档

### 2. 如果是文本/关键词 → 直接按7列标准表格分析输出

---

## 资料分析流程

### 准备阶段

1. 提取资料内容
2. 识别A股上市公司
3. 对识别结果进行去重和排序

### 分析阶段

对每个识别的A股上市公司，按以下7列格式逐一分析：

| 企业名称 | 产业链 | 细分环节 | 战略地位 | 核心竞争力 | 新增长极 | 推荐逻辑 |
|---------|-------|---------|---------|-----------|---------|------|

---

## 七列表格格式规范

### 第1列：企业名称
- **必填项**
- A股上市公司名称
- **此列为唯一登记依据** - 只有在此列出现的标的才视为正式登记标的
- 名称需加粗，使用Markdown标记：**公司名**

**示例：**
```markdown
**宁德时代**
```

### 第2列：产业链
- **必填项**
- 采用申万行业分类标准
- 格式：申万一级-申万二级-申万三级

**示例：**
```markdown
电力设备-电池-电池化学品
```

### 第3列：细分环节
- **必填项**
- 列出该公司的主要业务关键词
- 格式：关键词1\n关键词2\n关键词3

**示例：**
```markdown
动力电池
储能系统
电池材料
```

### 第4列：战略地位
根据企业在所属产业链中的市场地位、技术水平和行业影响力提炼。

**输出要求：**
- 一句精炼陈述（不超过50字）
- **排名类、地位类、唯一/稀缺类、份额类、突破类关键词需用Markdown加粗**

**加粗关键词类型：**
- 排名类：全球第一、全国领先、龙头等
- 地位类：核心供应商、关键技术等
- 唯一/稀缺类：唯一、独家、首家等
- 份额类：市场份额占比XX%、占比第一等
- 突破类：打破垄断、实现国产替代等

**示例：**
```markdown
全球动力电池装机量**第一**，市场份额超**30%**，**国内唯一**实现CTP技术量产的动力电池企业
```

### 第5列：核心竞争力
**格式要求（精简为1-3句，分号分隔）：**

1. 核心技术/市场要点
2. 国内主要竞争对手（格式：国内主要竞争对手为XX股份、XX科技）
3. 核心客户/合作商（格式：为XX股份等供应XX产品）

**示例：**
```markdown
全球领先的动力电池制造商；国内主要竞争对手为比亚迪、亿纬锂能；为特斯拉、蔚来等供应电池产品
```

### 第6列：新增长极
根据企业的第二增长曲线、新业务布局或跨界拓展方向提炼。

**输出格式：**
1. 先写新增长极定位描述（核心定位词需用Markdown加粗）
2. 再写"关联公司："，列出相关对标的A股公司名称并用Markdown加粗
3. 多个关联公司用顿号（、）分隔
4. 整个列内容用句号分隔不同方向

**示例：**
```markdown
**储能业务**成为第二增长曲线，已实现大型储能项目交付。关联公司：**宁德时代**、**阳光电源**
```

### 第7列：推荐逻辑
- 可选填
- 用于补充其他重要信息，如最新财务数据、政策利好、技术突破等

**示例：**
```markdown
2024年储能业务收入同比增长超100%
```

## 推荐逻辑列格式规范

在推荐逻辑列中，填写星级推荐等级和推荐逻辑：

### 星级评估标准

| 星级 | 标准 |
|------|------|
| ⭐⭐⭐⭐⭐ | 提及3次以上 + 投研强烈推荐 + 多个资料来源 + 业绩数据支撑 |
| ⭐⭐⭐⭐ | 提及2次或详细分析 + 投研推荐 + 有数据支撑 |
| ⭐⭐⭐ | 提及1次 + 有具体业务描述 + 投资逻辑清晰 |
| ⭐⭐ | 提及1次 + 业务关联度一般 |
| ⭐ | 提及1次 + 需进一步验证 |

### 推荐逻辑列格式

推荐逻辑列填写两行内容：   

```markdown
**推荐星级**：⭐⭐⭐⭐
**推荐逻辑**：逻辑内容描述
```

### 示例

```markdown
**推荐星级**：⭐⭐⭐⭐
**推荐逻辑**：投研重点分析，订单规模150亿元排至2027年，NOR Flash全球第三+MCU国内龙头，双轮驱动增长
```

### 填写规则

1. **推荐星级** → 根据提及次数、投研推荐程度、相关资料丰富度综合评估
2. **推荐逻辑** → 根据公司战略地位、核心竞争力、新增长极综合提炼，**推荐逻辑**四字加粗

## 注意事项

1. **A股上市公司认定**：仅识别在沪深交易所上市的公司，港股、美股、指数等不计入
2. **产业链分类**：采用申万行业分类标准，便于知识库的分类管理
3. **数据时效性**：注意资料的发布时间，优先使用最新的数据
4. **信息来源核实**：对于数据类信息，尽量核实其准确性
5. **格式一致性**：保持七列表格格式的一致性，便于后续的数据处理和分析
