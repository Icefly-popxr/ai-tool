# Skills 技能目录索引

## 📁 目录说明 (扩展目录)
自定义脚本、自动化工具、提示词模板

**说明**: Skills是在PARA基础上的扩展，用于存放自动化工具

## 📂 子目录
- `index/` - Skills索引目录
- `templates/` - 模板文件夹
- `scripts/` - 脚本文件夹

## 🛠️ 可用Skills

### 知识卡片锻造系统
- 激活词：知识卡片、生成卡片
- 用途：把零散经验提炼为结构化知识卡片
- 命令：/锻造、/迭代、/升级、/评估

### 灵感炼金炉
- 激活词：灵感、我有一个灵感
- 用途：7刃切割产生锋利洞见
- 命令：「更深」「换角度」「更毒舌」

### 内容炼金
- 激活词：创建口播文案、生成口播
- 用途：素材转化为可拍摄口播逐字稿
- 命令：「诊断」「打磨」「对标」「上强度」「最小化」
