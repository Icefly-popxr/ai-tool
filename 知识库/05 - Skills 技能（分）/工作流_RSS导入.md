# RSS信息流自动导入工作流

> 自动从外部订阅源拉取内容 → 存入 Resources 参考资料

---

## 📋 完整流程

```
RSS订阅源（你关注的信息源）
    ↓
脚本自动拉取最新文章
    ↓
转换为Markdown格式
    ↓
存入 03 - Resources 参考资料/
    ↓
（可选）AI分析 → 识别经典概念 → 写入Wikis概念库
```

---

## 🛠️ 需要准备的

| 项目 | 说明 | 操作人 |
|------|------|--------|
| Python | 运行脚本的语言环境 | 你可能已安装 |
| feedparser库 | Python解析RSS的库 | 安装一次即可 |
| RSS订阅地址 | 你想关注的博客/新闻源 | 你自己收集 |
| 脚本文件 | 自动拉取+保存 | 我已帮你创建 |

---

## 🚀 配置步骤

### 1. 安装依赖（只需一次）

打开终端，运行：
```bash
pip install feedparser
```

### 2. 填写RSS订阅地址

我已在脚本中用 `# ← 替换成你的订阅地址` 标注了修改位置。

### 3. 手动运行

```bash
cd /d "e:/GitHub/TRAE SOLO/知识库"
python 05 - Skills 技能/脚本_rss收集.py
```

### 4. 设置定时自动运行（进阶）

Windows搜索"任务计划程序" → 创建基本任务 → 设置每周运行

---

## 💬 与AI配合

内容拉取到 Resources 后，告诉AI：

```
请扫描 03 - Resources 参考资料/ 中的新内容，识别其中的经典学术概念，提取到 06 - Wikis 概念库/concepts/。
```
