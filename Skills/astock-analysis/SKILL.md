# A股上市公司知识库分析技能

## 核心功能

本技能帮助用户从各种资料来源中提取A股上市公司信息，并按照标准化格式生成分析表格。

## 支持的资料类型

- 📄 网页链接（网页内容抓取）
- 🖼️ 图片（OCR识别）
- 📑 PDF文档
- 📝 Markdown文档
- 📊 文本内容

## 输出表格格式（7列）

### 第1列：企业名称
- **必填项**
- A股上市公司名称
- **此列为唯一登记依据** - 只有在此列出现的标的才视为正式登记标的
- 名称需加粗，使用Markdown标记：**公司名**

**示例：**
```markdown
**宁德时代**
```

### 第2列：产业链
使用**申万三级行业标准**进行分类。

**输出要求：**
- 格式：`申万一级-申万二级-申万三级`
- 如：电力设备-电池-电池化学品、电力设备-光伏设备-光伏加工设备

**参考示例：**
```markdown
电力设备-电池-电池化学品
```

**常见产业链参考：**
- 电力设备-电池-电池化学品（璞泰来、亿纬锂能）
- 电力设备-电池-锂电专用设备（赢合科技、先导智能）
- 电力设备-光伏设备-光伏加工设备（晶盛机电、连城数控）
- 电子-半导体-数字芯片设计（寒武纪、景嘉微）
- 汽车-汽车零部件-汽车电子（德赛西威、华阳集团）

### 第3列：细分环节
根据企业的主营业务和产品特点，提炼其细分环节的具体关键词。

**输出要求：**
- 用①-③个具体的技术/产品关键词
- **每个关键词另起一行**
- **不使用完整句子**
- **不使用修饰词**

**示例：**
```markdown
动力电池
储能系统
电池材料
```

### 第4列：战略地位
根据企业在所属产业链中的市场地位、技术水平和行业影响力提炼。

**输出要求：**
- 一句精炼陈述（不超过50字）
- **排名类、地位类、唯一/稀缺类、份额类、突破类关键词需用Markdown加粗**

**加粗关键词类型：**
- 排名类：全球第一、全国领先、龙头等
- 地位类：核心供应商、关键技术等
- 唯一/稀缺类：唯一、独家、首家等
- 份额类：市场份额占比XX%、占比第一等
- 突破类：打破垄断、实现国产替代等

**示例：**
```markdown
全球**动力电池装机量第一**，**市场份额超30%**，**国内唯一**实现CTP技术量产的动力电池企业
```

### 第5列：核心竞争力
**格式要求（精简为1-3句，分号分隔）：**

1. 核心技术/市场要点
2. 国内主要竞争对手（格式：国内主要竞争对手为XX股份、XX科技）
3. 核心客户/合作商（格式：为XX股份等供应XX产品）

**示例：**
```markdown
全球领先的动力电池制造商；国内主要竞争对手为**比亚迪**、亿纬锂能；为特斯拉、蔚来等供应电池产品
```

### 第6列：新增长级
根据企业的第二增长曲线、新业务布局或跨界拓展方向提炼。

**输出格式：**
1. 先写新增长级定位描述（核心定位词需用Markdown加粗）
2. 再写"关联公司："，列出相关对标的A股公司名称并用Markdown加粗
3. 多个关联公司用顿号（、）分隔
4. 整个列内容用句号分隔不同方向

**示例：**
```markdown
**储能业务**成为第二增长曲线，已实现大型储能项目交付。关联公司：**宁德时代**、**阳光电源**
```

### 第7列：备注（可选）
- 补充说明
- 如资料来源、关键时间节点、风险提示等
- 非必填项

**示例：**
```markdown
2024年储能业务收入同比增长超100%
```

## 内部链接规则（重要）

- 所有提及的公司名称，如果**在第1列"企业名称"中存在**，需加上Markdown锚点链接
- 链接格式：`[**公司名**](#公司名)`
- 如果公司不在知识库中，暂不加链接

**示例：**
如果表格中有"**宁德时代**"和"**比亚迪**"：
```markdown
为**特斯拉**、**蔚来**等供应电池产品
```
（特斯拉不是A股上市公司，不加链接）

```markdown
国内主要竞争对手为**比亚迪**、亿纬锂能
```
（比亚迪在第1列，加锚点链接）

**知识库中公司的锚点设置方式：**
在知识库文件中，每个公司信息块前添加对应标题：
```markdown
## 宁德时代 {#宁德时代}

| 企业名称 | 产业链 | ...
```

**注意事项：**
- 锚点名称必须与公司名称完全一致（包括特殊字符）
- 锚点链接中不能包含空格，使用`#公司名`格式

## 工作流程

### 1. 接收资料
- 如果是链接，使用WebFetch工具获取内容
- 如果是图片，使用OCR识别（如有工具支持）
- 如果是PDF/Markdown，直接读取内容
- 如果是文本，直接处理

### 2. 分析汇总资料（仅供用户参考，不写入知识库）
对资料进行深度分析，形成汇总报告，包括：
- **资料概要**：资料的来源、类型、主要内容
- **产业链全景**：资料涉及的产业链梳理
- **核心标的**：识别出的主要A股上市公司
- **关键洞察**：产业趋势、竞争格局、技术路线等
- **数据质量评估**：信息的完整性、准确性、可信度

### 3. 识别A股上市公司
- 提取所有提及的A股上市公司名称
- 注意识别股票代码（如000001）和公司全称/简称

### 4. 按列分析
对每个识别的A股上市公司，按7列格式逐一分析：

```
| 企业名称 | 产业链 | 细分环节 | 战略地位 | 核心竞争力 | 新增长极 | 备注 |
|---------|-------|---------|---------|-----------|---------|------|
| **公司名** | 申万一级-申万二级-申万三级 | 关键词1<br>关键词2<br>关键词3 | 精炼陈述（加粗关键词） | 核心要点；竞争对手；核心客户 | 定位描述。关联公司：**公司A**、**公司B** | 可选备注 |
```

### 5. 展示结果
- **先展示分析汇总**（仅供用户参考）
- **再以Markdown表格形式展示分析结果**
- 前置"[待确认]"标记
- 等待用户输入

### 6. 确认并写入知识库
- **用户确认后才写入** - 只有用户明确回复"确认"或"加入"后，才将标的正式写入知识库
- **只写入表格内容** - 分析汇总内容不写入知识库
- 如果用户说"不要"、"删除"等，明确拒绝写入

**用户回复示例：**
- ✅ "加入" - 写入知识库（只写入表格）
- ✅ "确认" - 写入知识库（只写入表格）
- ✅ "好的" - 写入知识库（只写入表格）
- ❌ "不要" - 不写入
- ❌ "算了" - 不写入

```markdown
# A股上市公司知识库

| 企业名称 | 产业链 | 细分环节 | 战略地位 | 核心竞争力 | 新增长级 | 备注 | 录入时间 |
|---------|-------|---------|---------|-----------|---------|------|---------|
| [**宁德时代**](#宁德时代) | 电力设备-电池-电池化学品 | 动力电池<br>储能系统<br>电池材料 | 全球**动力电池装机量第一**，**市场份额超30%** | 全球领先的动力电池制造商；国内主要竞争对手为[**比亚迪**](#比亚迪)、亿纬锂能；为特斯拉、蔚来等供应电池产品 | **储能业务**成为第二增长曲线。关联公司：[**比亚迪**](#比亚迪)、[**阳光电源**](#阳光电源) | 2024年储能业务收入同比增长超100% | 2026-05-08 |
```

## 注意事项

1. **准确性优先** - 只输出有明确资料支撑的信息，不猜测
2. **简洁精炼** - 遵循各列的格式要求，不添加冗余信息
3. **专业术语** - 使用准确的产业链和技术术语
4. **客观中立** - 客观描述企业情况，不添加主观评价
5. **及时更新** - 如果知识库中已有该公司，分析后标注"更新"而非"新增"

## 错误处理

- 如果资料中无法提取有效信息，明确告知用户
- 如果某列信息缺失，标注"[信息不足]"，不编造信息
- 如果识别到的不是A股上市公司（如港股、美股、上市公司子公司），跳过并在回复中说明
